AutoCAD Customize Tool Components. バージョンアップ情報

● Version 1.0 -----------------------------------------------------

1) Delphi2.0/3.1対応　TAcadControl コンポーネント公開

● Version 1.1 -----------------------------------------------------

1) Delphi4.0対応　TAcadControl コンポーネント作成

● Version 1.2 -----------------------------------------------------

1) Delphi4.0対応と機能追加　TAcadControl コンポーネント公開
2) AutoCAD LT98 対応

● Version 1.3 -----------------------------------------------------

1) SelectDxf, SelectAtt メソッドの追加
2) ActiveX 版との調整のため、ReSize メソッドの名前を AcReSize に変更

● Version 1.31 ----------------------------------------------------

1) OnWaitMessage イベントが発生しない点を修正
2) SelectDxf, SelectAtt メソッドでファイル作成完了まで待機

● Version 1.40 ----------------------------------------------------

1) ActiveX 版不具合の修正のため、コマンド送信ロジックの改良(AcadUnit)
2) プロパティの追加とバージョン情報付加

● Version 1.50 ----------------------------------------------------

1) Delphi4.0のみに対応とし、テストバージョンとして公開
2) AutoCAD2000に対応
3) AcadControl だけでなく、AutoCADカスタマイズ用ツールを統合して
   AcadToolパッケージを提供
4) コンポーネント類は、ソースの公開

● Version 1.51 ----------------------------------------------------

1) Delphi3.1/4.0/5.0対応とし、テストバージョンとして公開
2) AutoCAD LT2000(販売前)に対応(製品が販売される前の予想設定)
3) AcadUnit ユニットの全面調整
   TAcadWindow 構造体を破棄し、TAcadObject クラスとして、
   オブジェクト指向に沿った構造に調整し直した。
4) AutoCAD の EXEファイルのバージョンをチェックするようにした。
5) ダイアログの変化イベントを遅延させた。

● Version 1.52 ----------------------------------------------------

1) Delphi3.1/4.0/5.0 対応公開
2) AutoCAD LT2000 に正式対応
3) AcadUnit ユニット内の MDI図面ウィンドウ の検索処理変更
4) AcadUnit ユニット内の MDILock 処理変更
5) AcadUnit ユニット内の コマンド送信ロジックを変更し、さらに安定化
6) ダイアログの変化イベントをさらに遅延させた。

● Version 1.53 ----------------------------------------------------

1) コマンド送信処理変更
2) ダイアログの取得失敗を改善
3) VoloViewEx コンポーネントの追加

● Version 1.54 ----------------------------------------------------

1) VoloViewEx コンポーネントで日本語2000-083と英語2000-129の共用化
2) AcadControlにメニュークリックを行うメソッドを追加

● Version 1.55 ----------------------------------------------------

1) OnProcessLoopイベントとProIndexメソッドを追加
2) AcadControlX(ActiveX版)の製作による全体見直し
3) Ver1.5正式バージョンの修正公開

● Version 1.60 ----------------------------------------------------

1) AutoCAD2000i, AutoCAD LT2000i サポート
2) Windows Me, Windows2000 サポート
3) AutoCADの各バージョン処理を再整理(AcadVInf)
4) 汎用関数群の再チェックと整理(AcadFunc)
5) AcadControl にバージョン毎管理用関数とプロファイル処理追加

● Version 1.61 ----------------------------------------------------

1) C++Builder5用パッケージ, ActiveXコントロール対応
2) 上記対応に伴い、ソースコードの変更
3) GetLastFocus メソッドの不具合修正

● Version 1.70 ----------------------------------------------------

1) AutoCAD2002, AutoCAD LT2002 サポート
業務にて必要だったため急きょ作成したが、一般は非公開

● Version 1.80 ----------------------------------------------------

1) AutoCAD2002, AutoCAD LT2002 公式サポート
2) Delphi6 パッケージ対応
3) その他、詳しくはヘルプに明記

● Version 1.81 ----------------------------------------------------

1) RunScript関数のヘルプ修正
2) Close メソッドの上書き保存不具合修正
3) TOpenDwgDialog, TSaveDwgDialog の定義ユニット変更と不具合修正

● Version 1.82 ----------------------------------------------------

1) ActiveX版でロックする問題の修正但し改善されなかった

● Version 1.90 ----------------------------------------------------

1) AutoCAD2004, AutoCAD LT2004 サポート
2) Delphi7 パッケージ対応 / Delphi4 対応破棄
3) その他、詳しくはヘルプに明記

● Version 1.91 ----------------------------------------------------

1) GetPoint などを繰り返し実行した際にFalseが返る問題の修正
2) WaitCommand で別のAutoCADに切り替わった際に関数が戻らない問題の修正
