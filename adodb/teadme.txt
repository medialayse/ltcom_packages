このプログラムは AutoCAD の LISPプログラム内で ADO を使いデータベースの
操作ができるようにするLISP関数群を提供するプログラムです。


■ 対応ＣＡＤ

  AutoCAD 2000, AutoCAD 2000i, AutoCAD 2002 のレギュラー版か、
  AutoCAD LT2000, AutoCAD LT2000i と LT/com2000i Ver1.20以降
  AutoCAD LT2002 と LT/com2002 Ver1.30以降
  (ＬＴ版で使うためには、別途 LT/com2002 を購入する必要があります)


■ 動作環境

  ADODB は 対応ＣＡＤとＡＤＯを使うためのドライバが必要になります。
  以下のドライバが無いと動作いたしません。
    Windows95 ... DCOM95, MDAC2.0以降
    Windows98, Me, NT4.0, 2000 ... MDAC2.0以降

  DCOM for Windows 95 Version 1.3 日本語版
    URL : http://www.microsoft.com/japan/com/dcom/dcom95/dcom13.asp
    上記のURLからダウンロードしてください。
  MDAC(Data Access Components SDK)
    URL : http://www.microsoft.com/japan/download.htm
    製品名に "Data Access Components SDK" 、オペレーティング システムを
    お使いのものを選択して、ダウンロードしてください。


■ インストール方法

  AutoCAD 2000, 2000i, 2002 で使う場合...
    アプリケーションロード(APPLOAD)コマンドで、ADODB2L.arx を
    ロードしてください。

  AutoCAD LT2000, LT2000i, LT2002 で使う場合...
    アプリケーションロード(APPLOAD)コマンドで、ADODB2L.drx を
    ロードし、自動ロードに設定してください。
    (APPLOADコマンドは、LT/com2002 を起動中に使えます)

  関数個別についてはヘルプを参照してください。
